## 📖 About

React Stopwatch is a stopwatch application built using **React** and **Parcel**. It allows users to start, stop, and reset a timer while displaying elapsed time in **hours, minutes, and seconds**.

This project was built to strengthen my understanding of **React Hooks**, timer management with JavaScript, and real-time UI updates.

---

## ✨ Features

* ⏱️ Start the stopwatch
* ⏸️ Pause the timer
* 🔄 Reset the timer
* 🕒 Display time in **HH:MM:SS** format
* ⚡ Real-time updates every second
* ⚛️ Built with React Hooks

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/aryankapoor-dev/React-Stopwatch.git
```

Navigate to the project directory

```bash
cd React-Stopwatch
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm start
```
